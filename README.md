ofxHomography
=============
![ofxHomography](https://github.com/paulobarcelos/ofxHomography/raw/master/screenshots/example.png)
Introduction
------------
Openframeworks helper addon with functions for finding the homography between two sets of points.

License
-------
MIT

Installation
------------
Just drop the folder into the `openFrameworks/addons/` folder.

Dependencies
------------
N/A

Compatibility
------------
Tested with openFrameworks 0.12.1 on macOS.

The example keeps the official openFrameworks `Makefile` workflow and also includes an `of.yml` recipe for ofWorks/ofGen users.
